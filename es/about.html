Acerca de GoldenDog

Acerca de GoldenDog

GoldenDog Linuxes una Estación de Trabajo Linux que pretende ser amigable, simple de usar y, al mismo tiempo, proporcionar toda la integridad, estabilidad y fiabilidad que una plataforma como Debian Stable tiene para ofrecer.



¿Por qué Debian?

Hubo algunas razones por las cuales. Principalmente porque he sido usuario de Debian durante la mayor parte de mi vida, comenzando alrededor de 2004, y lo usé como mi sistema principal hasta el año 2019, cuando comencé a usar Fedora también. Estos 15 años de experiencia me hicieron muy familiar con cómo funciona este sistema operativo.
GoldenDog no es el único trabajo que utiliza Debian como plataforma. Muchos sistemas populares como Ubuntu, Mint y Kali también lo utilizan. Esto se debe a que, en su núcleo, está muy meticulosamente planificado y mantenido, lo que lo convierte en una excelente opción para un motor fiable que se adapta a casi cualquier caso de uso.

Debian también fomenta trabajos derivados que usan su software para usar sus espejos oficiales. Es por eso que GoldenDog ejecuta exclusivamente software y espejos oficiales de Debian, aprovechando las frecuentes actualizaciones y parches que Debian proporciona, especialmente porque este es un proyecto de una sola persona en el que trabajo cuando el tiempo lo permite, y no podría empaquetar miles de paquetes .deb yo sola de forma regular. Usar solo paquetes oficiales de Debian hace feliz a todos.

Detalles Técnicos

Requisitos Mínimos

  • Memoria: 2GB
  • Núcleos de CPU: 2
  • Espacio en Disco: 50GB

Requisitos Recomendados

  • Memoria: 8GB
  • Núcleos de CPU: 4
  • Espacio en Disco: 120GB

Soporte de Video

GoldenDog utiliza los módulos de controladores de Debian, por lo que está disponible el soporte tanto para GPUs NVIDIA como AMD. En una ventana de terminal, escribe:

Para NVIDIA: sudo gdl install nvidia-driver

Para AMD: sudo gdl install amdgpu

Soporte para Snap y Flatpak

Por defecto, ni flatpak ni snap están instalados. Se pueden instalar fácilmente a través de gdl/apt desde una ventana de terminal.

Para Snapd: sudo gdl install snapd

Para Flatpak: sudo gdl install flatpak

Gestor de paquetes gdl/apt

Actualmente, GDL utiliza el gestor de paquetes APT. En este momento, gdl es simplemente un alias para apt. Puedes usarlos indistintamente.

goldendog-update

Este comando es una interfaz de usuario de texto que lista los paquetes que podrían ser actualizados y te permite seleccionarlos individualmente. Podría ser útil cuando no quieras realizar una actualización completa y seleccionar paquetes específicos en su lugar, por ejemplo, cuando deseas evitar actualizar a una cierta versión del kernel o cualquier otra cosa.
