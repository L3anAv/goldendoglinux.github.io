Una estación de trabajo Linux ¿hecha simple?

GoldenDog Linux es una selección automatizada de paquetes y configuraciones para un sistema operativo Debian Stable. Esta es la selección de paquetes que diseñé para uso personal y ahora he decidido automatizar y empaquetar como un archivo ISO instalable.

¿Qué pasa con actualizar mi sistema?

Como GDL, en su núcleo, utiliza software de Debian, y Debian recomienda usar sus espejos para garantizar la fiabilidad y estabilidad, GoldenDog utiliza los repositorios oficiales de Debian Stable, manteniendo un rendimiento robusto y fiable. Además, todas las actualizaciones y bibliotecas de software que están disponibles actualmente para Debian Bookworm también están disponibles para GoldenDog.

Un escritorio lindo para el uso diario

GoldenDog no depende de flatpak o snap para el software. Viene con algunos programas preinstalados, como Visual Studio Code, junto con su repositorio oficial (Microsoft) para que siempre puedas mantenerlo actualizado a su última versión. ¡Muchas aplicaciones se ejecutan de forma nativa como Steam, kdenlive, obs, Spotify y muchas más! ¡Ideal para trabajar, jugar, desarrollar, revisar el correo y chatear!

GDL nació como una herramienta de automatización para que yo misma desplegara mi propio sistema operativo de manera consistente en mis diferentes máquinas. Algunas personas lo escucharon y sugirieron que convirtiera mi herramienta en una ISO pública instalable para cualquiera que quiera tener una configuración similar, así que aquí está. Ha sido hecho a mano con mucho amor. ¡Espero que les guste!

Alexia.
